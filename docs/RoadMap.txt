PHASE 9 — MICROCOSMOS MODE (FULLY SPECIFIED)

		Identity:
			Granular reinterpretation engine that feeds into
			Big Pi diffusion + tail.
			Separate engine, not embedded in reverb tank.

		Signal Flow:

			Input
				↓
			Capture Buffer (2.0s stereo @ 48kHz)
				↓
			Grain Engine (~32 voices)
				↓
			Soft Saturation (optional)
				↓
			Diffusion Network (reuse existing)
				↓
			Space (reuse Tank tail)
				↓
			Output

		Capture Buffer:
			96,000 samples per channel
			192,000 floats total
			~768 KB memory

	Grain Model:

		struct Grain
		{
		float position;
		float speed;
		float length;
		float age;
		bool active;
			};

	MAX_GRAINS = 32

		Grain Processing Per Sample:

		Linear interpolation read

		position += speed

	Hann window:
		w(t) = 0.5 * (1 - cos(2πt))

		Sum to wet

		age++

		deactivate when age >= length

	Pitch:
		speed > 1.0 → pitch up
		speed < 1.0 → pitch down
		speed < 0 → reverse

	Internal Behaviors:

		Mosaic:

			Random start positions

			Slight pitch variation

			Medium density

		Tunnel:

			Slow grains (<1.0 speed)

			Long grains

			Heavy diffusion

			Drone behavior

		Sequence:

			Grid-triggered grains

			Rhythmic feel

			Future MIDI clock support

	Macro Mapping:

			Decay → Grain length
			PreDelay → Grain density
			ModDepth → Random pitch range
			ModRate → Grain trigger rate
			Mix → Dry/Wet
			Tone → Grain low-pass
			Size → Capture buffer scan range

		MVP Order:

			Capture buffer

			Single grain + Hann window

			Multiple grains + scheduler

			Add sub-modes

			Route through diffusion + tail
